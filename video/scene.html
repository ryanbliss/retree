04 · view models
ReactiveNode decorators enable performance precision.
task-board.tsReactiveNode

@select

emits only when the getter’s value changes
tasks[2].done = true→doneCount 1→2 — emits
tasks[2].title = "…"→doneCount 2 — quiet ∅

@memo

caches until its dependencies change
read results→cache hit · no recompute
query = "tree"→recomputes · ×1

@ignore

reads and writes without emission
scratch.draftClicks++→no emission ∅
scratch.draftClicks += 3→emissions: 0
